〜初めて遊ぶ方へ〜
　初めてBlind Justiceをプレイする方は、TOPの「説明書」に目を通してください。
「説明書」は数あるBlind Justice共通のものです。この鯖と異なる点もあります。
「wiki」はこの鯖専用のものです。「説明書」とゲームでことなる点は、こちらで確認してください。
「説明書」「wiki」の両方を読むと、たいていの疑問は解決します。
どちらに目を通しても解決しない場合、各国の「作戦会議室」で質問するのをお勧めします。
また、「説明書」の方に要望等書いても、ゲームには反映されません。


〜禁止事項〜
・連打
ｻｰﾊﾞｰからの反応が重いときにボタン連打する人がいますが、データ破損の原因となります。
ﾍﾟｯﾄが変ったり（預かり所を操作しているときに連打するとよく起こります）、
ｷｬﾗﾃﾞｰﾀが全て失われた例もあります。ﾃﾞｰﾀが失われても対応できません。
サーバーの反応が悪い時は、落ち着いて待ってください。
・サーバーに負荷をかける行為
このサーバーは有料のサーバーを使用しています。
故意にサーバーに不可をかける行為によりサーバーが落ちた場合、
このゲームが落ちるだけならまだいいですが、他の契約者にまで迷惑がかかる場合があります。
絶対に止めてください。
また、そのような行為があるとわかった場合、ｷｬﾗﾃﾞｰﾀ削除の上、アクセス禁止処理を行います。
